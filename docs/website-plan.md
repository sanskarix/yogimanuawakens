# Website Plan
# Yogi Manu

Version: 1.0

---

# Vision

This website is not a marketing website.

It is not a portfolio.

It is not a link tree.

It is a peaceful digital sanctuary where visitors can slow down, breathe, explore, practice and reconnect.

Every page should feel intentional.

Every animation should create calm.

Every interaction should encourage presence rather than urgency.

The website should become the central home for everything Yogi Manu shares:
- Yoga
- Kirtan
- Spiritual Teachings
- Neem Karori Baba devotion
- Hanuman devotion
- Events
- Daily Practice
- Community

---

# Primary Goals

• Unify all content under one home.

• Create a timeless online presence.

• Help new visitors understand Yogi Manu.

• Encourage daily practice.

• Organically grow YouTube audiences.

• Make donations feel natural.

• Prepare for future retreats, memberships and online courses.

---

# Emotional Goals

Visitors should leave feeling

- Peaceful
- Grounded
- Curious
- Hopeful
- Connected

Never overwhelmed.

Never marketed to.

# RESPONSIVE RULES

Desktop

Large whitespace.

Maximum content width around 1200–1300px.

Comfortable reading width.

Tablet

Reduce spacing.

Maintain hierarchy.

Mobile

Single-column layouts.

Large touch targets.

No tiny typography.

Fast loading.

---

# PERFORMANCE

Lighthouse score above 95.

Lazy-load all media.

Use optimized images.

Avoid unnecessary JavaScript.

Prefer server components where possible.

---

# ACCESSIBILITY

Keyboard navigation.

Visible focus states.

Reduced motion support.

High contrast.

Readable font sizes.

Descriptive alt text.

Semantic HTML.

"Does this help someone become a little more peaceful, a little more present, or a little more connected?"

If not, it does not belong on the website.
